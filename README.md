# 產業龍頭儀表板

整合台股 + 全球產業分類儀表板，單檔自包式 HTML。

## 資料來源（2026-05-05 快照）

- **macromicro 產業情報平台**：11 產業 × 71 子產業 × 450 全球代表廠商
- **FinLab API**：台股全市場 2318 檔 + 36 官方產業 + 還原權值漲跌幅
- **twstockflow**：74 個族群主題

## 功能

- 三種分組視角切換：macromicro 子產業 / FinLab 產業 / twstockflow 主題
- 7 個期間漲跌幅：天 / 周 / 月 / 季 / 半年 / 1Y / 3Y
- 產業 section 顯示中位數漲跌（看大方向）
- 點 tag 自動切換對應分組
- 排序：市值 or 漲跌幅

## 注意

資料為靜態快照，非即時。
